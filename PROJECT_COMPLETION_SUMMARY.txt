🚀 快速開始
═══════════════════════════════════════════════════════════════════════════════

1. 進入項目目錄:
   cd /workspaces/fraud-prevention

2. 安裝依賴 (已完成):
   npm install

3. 啟動開發伺服器:
   npm run dev

4. 打開瀏覽器:
   http://localhost:5173

5. 開始遊戲!
   • 在掃描儀輸入詐騙訊息
   • 進入戰鬥並選擇反擊卡牌
   • 瀏覽圖鑑與懸賞任務
