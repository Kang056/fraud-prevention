# 詐獸獵人 (Project S.C.A.M.)

> 🎮 **全球首款沉浸式防詐 RPG 遊戲化平台**

在充滿雜訊的數位荒野中，狩獵惡意，反殺騙局。

## 🎯 快速開始

```bash
# 安裝依賴
npm install

# 啟動開發伺服器 (http://localhost:5173)
npm run dev

# 構建生產版本
npm run build
```

## 📚 核心功能

### 🎯 戰術掃描儀 (Scanner)
輸入可疑資訊進行風險評估，自動識別詐獸

### ⚔️ 詐獸殲滅戰 (Battle)
3 選 1 的反擊機制，在 10 秒內做出正確選擇擊敗詐獸

### 📖 獵人圖鑑 (Bestiary)
收集擊敗的詐獸卡牌，瀏覽詐騙手法與防禦知識

### 💰 懸賞公會 (Bounty)
貢獻詐騙樣本，獲得獎勵並被永久記錄在遊戲中

## 🎨 視覺風格

- **賽博龐克像素風** - 復古掌機 + 霓虹光效
- **故障藝術特效** - 動態掃描線與視覺擾亂
- **移動優先設計** - 全面響應式支持

## 🎮 遊戲流程

```
掃描儀 → 輸入資訊 → 風險評估 → 詐獸出現 → 進入戰鬥
           ↓
        安全 → 返回掃描儀
        危險 ↓
           選擇反擊卡牌 (10秒)
           ├─ 陷阱卡 → 💀 GAME OVER
           ├─ 防禦卡 → 🛡️ DRAW
           └─ 爆擊卡 → ⚔️ VICTORY → 收錄詐獸 + 獎勵
```

## 💡 特色機制

### 詐獸屬性系統
- 🤑 **貪婪系** - 利益驅動型詐騙
- 😨 **恐懼系** - 恐慌製造型詐騙
- 💔 **情感系** - 感情利用型詐騙

### 風險等級
🟢 低 | 🟡 中 | 🔴 高 | ☠️ 極限

### 獎勵系統
- 經驗值 (XP) - 提升獵人等級
- 金幣 (Gold) - 購買與兌換
- 成就 - 稱號與勳章
- 認可度 - 永久記錄在詐獸卡牌
