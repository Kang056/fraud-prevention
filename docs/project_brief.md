# Project S.C.A.M. (The Scam Hunter) - Project Brief

**Core Slogan:** Hunting malice and counter-killing scams in the noisy digital wilderness.

## 1. Project Overview
*   **Positioning:** The world's first "Immersive Anti-Fraud RPG Gamified Platform".
*   **Target Audience:** 15-45 years old digital natives and immigrants.
*   **Solution:** Transform "verification" into "battle", and "education" into "game achievements".

## 2. World & Art
*   **Lore:** 20XX, Cyberia Continent. "Glitch Monsters" (Scam Beasts) born from human greed/fear. User = Hunter, AI = Navi (Navigator).
*   **Visuals:** Cyberpunk Pixel Art. Retro neon (Purple, Cyan, Black), Glitch effects.

## 3. Core Game Loop
1.  **Scan:** Input suspicious info.
2.  **Battle:** "Extermination Mode". Micro-management turn-based battle.
3.  **Collect:** Record defeated monsters in Bestiary.
4.  **Bounty:** Report new scams for points/rank.

## 4. Game Mechanics

### A. The Scanner (Home)
*   Radar scan interface.
*   Input URL/Text/Screenshot.
*   **Safe:** Green, pleasant sound.
*   **Danger:** Red alert, Glitch effect -> Forced Battle.

*   Dynamic cards.
*   Attributes: Greed, Fear, Emotion.

### D. The Bounty Guild
*   Wanted list.
*   First Blood mechanism for reporting new scams.

## 5. Tech Stack & AI
*   **Validation:** Layer 1 (Database: 165, Whoscall, Google Safe Browsing), Layer 2 (LLM: Intent analysis -> Monster generation -> Card generation).
*   **Frontend:** React (SPA).
*   **Visuals:** Phaser.js or Three.js for effects. Vanilla CSS for styling (Cyberpunk).

## 6. Growth Strategy
*   Shareable Battle Result (IG Story style).
*   Raid Events (Global Boss).
