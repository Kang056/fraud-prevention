## 📋 項目概述

### 核心定位
全球首款**沉浸式防詐 RPG 遊戲化平台**，將被動查證轉化為主動娛樂，讓用戶通過遊戲學習反詐知識。

### 標語
> 在充滿雜訊的數位荒野中，狩獵惡意，反殺騙局。

### 目標受眾
15-45 歲數位原住民，特別是習慣滑手機、容易接觸網路詐騙的族群

## 🎮 遊戲機制

### 詐獸屬性系統
```
🤑 貪婪系 (Greed)     - 利益驅動型詐騙
😨 恐懼系 (Fear)      - 恐慌製造型詐騙
💔 情感系 (Emotion)   - 感情利用型詐騙
```

### 風險等級
```
🟢 低 (Low)          - 安全無虞
🟡 中 (Medium)       - 需保持警惕
🔴 高 (High)         - 明確風險
☠️ 極限 (Extreme)    - 高度危險
```

### 戰鬥機制
```
🔴 陷阱卡 (Trap)      → 💀 GAME OVER → 0 XP
🟡 防禦卡 (Defend)    → 🛡️ DRAW      → 20 XP
🟢 爆擊卡 (Critical)  → ⚔️ VICTORY   → 100 XP
```

### 獎勵系統
- **經驗值 (XP)** - 提升獵人等級
- **金幣 (Gold)** - 購買與兌換
- **成就徽章** - 永久成就記錄
- **認可度** - 獵人 ID 記錄在詐獸卡牌
