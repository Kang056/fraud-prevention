## 🚀 5 分鐘快速開始

### 1️⃣ 檢查環境
```bash
# 確保已安裝 Node.js 16+
node --version
npm --version
```

### 2️⃣ 進入項目目錄
```bash
cd /workspaces/fraud-prevention
```

### 3️⃣ 安裝依賴 (如未安裝)
```bash
npm install
```

### 4️⃣ 啟動開發伺服器
```bash
npm run dev
```

### 5️⃣ 打開瀏覽器
訪問 `http://localhost:5173`

## 🎮 遊戲操作指南

### 掃描儀頁面
1. 在文本框輸入可疑訊息或網址
2. 點擊 🔍 **開始掃描**
3. 等待 AI 分析 (模擬 1.5 秒)
4. 如果判定為高風險，點擊 ⚔️ **進入戰鬥**

### 戰鬥頁面
1. 詐獸出現並說出攻擊台詞
2. 在 **10 秒倒計時** 內選擇一張卡牌:
   - 🔴 **陷阱卡**: 錯誤選擇 → GAME OVER
   - 🟡 **防禦卡**: 安全選擇 → DRAW
   - 🟢 **爆擊卡**: 最佳反擊 → VICTORY ⭐
3. 查看結果與獎勵

### 圖鑑頁面
1. 查看已收集的詐獸卡牌
2. 使用篩選器快速查找
3. 點擊卡牌查看詳細信息

### 懸賞頁面
1. 瀏覽進行中的懸賞任務
2. 查看獎勵與難度等級
3. 了解參與方式

## 🔧 常見問題

### Q: 頁面加載很慢?
A: 這是正常的，因為包含了完整的 CSS 特效。生產版本將優化。

### Q: 戰鬥倒計時看不清?
A: 倒計時顯示在卡牌上方的圓形進度圈中，也有秒數數字顯示。

### Q: 如何重置遊戲?
A: 刷新頁面即可重置所有遊戲狀態。

### Q: 為什麼所有詐騙都能被掃描儀識別?
A: 這是 MVP 版本，使用模擬 AI。真實版本需要集成 LLM API。
